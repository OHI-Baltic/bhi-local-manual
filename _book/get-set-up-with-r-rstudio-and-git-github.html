Chapter 3 Get set up with R +Rstudio and Git +Github

Making OHI+ assessments in the Baltic Sea requires coding and using data science software.
You can learn this in OHI’s Intro to Open Data Science training book, where you you will learn a workflow with R, RStudio, Git, and GitHub.
